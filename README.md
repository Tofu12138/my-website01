# TocaFunGames Website

Static website for `tocafungames.com`, designed for GitHub Pages.

## App Store URLs

- Support URL: `https://tocafungames.com/`
- Marketing URL: `https://tocafungames.com/`
- Privacy Policy URL: `https://tocafungames.com/privacy/`
- Privacy Choices URL: `https://tocafungames.com/privacy/choices/`

## GitHub Pages deployment

1. Create a GitHub repository named `tocafungames.github.io` or use an existing repository.
2. Upload all files in this folder to the repository root on the `main` branch.
3. In GitHub, open **Settings** > **Pages** and choose **Deploy from a branch**, then select `main` and `/(root)`.
4. Under **Custom domain**, enter `tocafungames.com` and enable **Enforce HTTPS** after DNS verification completes.
5. At your domain provider, create four `A` records for `@`: `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, and `185.199.111.153`. Create a `CNAME` record for `www` pointing to `tocafungames.github.io`.

Before using the listed contact email addresses, configure email forwarding or mail hosting for `support@tocafungames.com` and `privacy@tocafungames.com`.
